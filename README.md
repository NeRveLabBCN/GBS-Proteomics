# Guillain-Barré Syndrome Longitudinal Proteomics (SomaScan 7k)

This repository contains the dataset and reproducible analysis pipelines for the longitudinal proteomic profiling of Guillain-Barré syndrome (GBS). The study utilizes the SomaLogic SomaScan 7k platform to identify differentially abundant proteins and enriched biological pathways across different disease stages.

## Study Design

Plasma samples were analyzed to compare proteomic signatures across three groups:
* **GBS Acute:** 20 samples taken during the acute phase of the disease.
* **GBS Recovery:** 20 longitudinal matched samples taken at 1-year follow-up.
* **Healthy Controls:** 15 matched healthy controls.

## Repository Structure

The repository is organized into data, reproducible analysis scripts, and statistical results.

### Data
* `Data/GBS_Proteomics_Raw_Github.csv`: The cleaned, anonymized abundance data (raw Relative Fluorescence Units). The proprietary `.adat` headers have been stripped to maximize open-source accessibility. Protein columns are indicated strictly by their SomaLogic IDs (`seq.XXXX.XX`).
* `Data/SomaScan_Protein_Dictionary_Github.csv`: A mapping dictionary linking each `seq.` ID to its corresponding Target protein name, Entrez Gene Symbol, and UniProt ID.

### Scripts
* `01_Limma_Analysis.R`: Performs `log2` transformation and differential abundance analysis using linear models (`limma`). Includes paired analysis (Acute vs. Recovery) and covariate-adjusted models for Age and Sex (Acute GBS vs. Healthy Controls, Recovery GBS vs. Healthy Controls). Generates Volcano Plots.
* `02_GSEA_Analysis.R`: Performs Gene Set Enrichment Analysis (`fgsea`) on the `limma` outputs using Hallmark, Reactome, and GO:BP databases. Generates normalized enrichment dot plots.

### Results
The output files generated by the scripts are provided here for immediate review:
* `limma_acute_vs_recovery.csv`: Full annotated differential abundance results between acute GBS and recovery GBS (1-year samples).
* `limma_acute_vs_healthy.csv`: Full annotated differential abundance results between acute GBS and healthy controls.
* `limma_recovery_vs_healthy.csv`: Full annotated differential abundance results between recovery GBS and healthy controls.
* `fgsea_AR_t.csv`: Pathway enrichment tables for acute vs recovery GBS.
* `fgsea_AC_t`: Pathway enrichment tables for acute GBS vs healthy controls.
* `fgsea_RC_t`: Pathway enrichment tables for recovery GBS vs healthy controls.

## Usage and Reproducibility

To reproduce this analysis locally:
1. Clone or download this repository.
2. Ensure your R working directory is set to the main repository folder.
3. Install the required R packages (`limma`, `fgsea`, `msigdbr`, `dplyr`, `ggplot2`, `patchwork`, `writexl`, `ggrepel`, `viridis`).
4. Run `01_Limma_Analysis.R` followed by `02_GSEA_Analysis.R`. The scripts use relative file paths to automatically pull from the `Data/` folder and apply the dictionary mapping dynamically.
